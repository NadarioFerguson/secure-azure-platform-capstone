# Secure Azure Platform Capstone

Production-grade Azure platform demonstrating Cloud Security and DevSecOps capabilities.

## Scope
- Secure hub-spoke networking
- Terraform-managed infrastructure
- Identity & access control (RBAC + Managed Identities)
- Secure CI/CD with policy and security gates
- Monitoring, detection, and incident response

## Repository Structure
- terraform/ – Infrastructure as Code (bootstrap, modules, environments)
- ci-cd/ – CI/CD pipelines and security gates
- diagrams/ – Architecture diagrams and threat models
- monitoring/ – Logging, alerts, and detection queries
- incident-response/ – Incident response playbooks and tabletop scenarios
